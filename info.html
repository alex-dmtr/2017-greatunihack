Ocado Technology

#ocado-technology

Design a 3D packing algorithm that uses a list of grocery items to calculate the number of containers required for a customer order, grouping the items into specific containers according to a set of predefined rules. The output will also contain a list of picking sequences which will describe which items from which order will go into which container based on the rules provided. More info Here.

The Hut Group

#the-hut-group

Build a tool to help customer service agents at THG prioritise and categorise customer emails, using NLP tools/techniques. Is the customer making a complaint? Is it a question about a product? Which of our websites are they talking about? Use any libraries/APIs you like. Should be able to analyse at least English, with other languages optional.
